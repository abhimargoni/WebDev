About HTML

 
    <!DOCTYPE> tag - This tag defines the document type.
    <html>  tag - This tag is used to define HTML document.
    <head>  tag - This tag is used to define header for the HTML document. Title and Internal CSS lies inside this tag.
    <title>  tag - This tag is used to define title for the HTML document.
    <body>  tag - This tag is used to define Body of the HTML document. Almost everything lies inside this tag.
    <p>  tag- This tag is used to define paragraph for the HTML document.
    <h1>  tag - This tag is used to define biggest heading inside the HTML document.
    <h2>  tag - This tag is used to define 2nd biggest heading.
    <h3>  tag - This tag is used to define 3rd biggest heading.
    <h4>  tag - This tag is used to define 4th biggest heading.
 tag - This tag is used to define 5th biggest heading.
    <h6>  tag - This tag is used to define smallest heading.
    <style>  tag - This tag is used to define internal CSS for any element inside the HTML document.
    <br>  tag - This is an empty tag (empty tags don't need to be closed) and is used to insert a line break.
    <hr>  tag - This tag is used to insert horizontal line.
    <ul>  tag - This tag is used to insert unordered list.
<ol>  tag - This tag is used to insert ordered list.
<li>  tag - This tag is used to insert list inside 
     or 
       tag.
    <strong>  tag - This tag is used to make the text inside it bold and gives importance to the text.
    <b>  tag - This tag is used to make the text inside it bold but doesn't give importance to the text.
    <i>  tag - This tag is used to make the text inside it italic.
    <u>  tag - This tag is used to underline the text inside it.
    <!--XXX -->  - This tag is used to insert comment.
<abbr>  tag - This tag defines an abbreviation.
<center>  - This tag aligns all elements inside it to the center.
<del>  - This tag defines text that has been deleted from the document.
<ins>  - This tag defines text that has been inserted into a document.
<mark>  - This tag highlights text.
<pre>  - This tag defines preformatted text.
<q>  - This tag defines quotation.
<sub>  - This tag defines subscripted text.
<sup>  - This tag defines superscripted text.
<time>  - This tag defines a date/time.
<form>  - This tag defines an HTML form for user input.
<input>  - This tag defines an input control.
<button>  - This tag defines a clickable button.
<textarea>  - This tag defines a multiple line input control.
<option>  - This tag defines an option in a drop-down list.
<iframe>  - This tag defines an inline frame.
<img>  - This tag defines an image.
<map>  - This tag defines an user side image-map.
<area>  - This tag defines an area inside an image-map.
<a>  - This tag defines a hyperlink.
<table>  - This tag defines a table.
<caption>  - This tag defines a table caption,
<th>  - This tag defines table header cell.
<tr>  - This tag defines a row in a table.
<td>  - This tag defines a cell in a table.
<meta>  - This tag defines metadata of the HTML document.
<footer>  - This tag defines a footer for the document.
